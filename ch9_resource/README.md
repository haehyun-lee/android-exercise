# 메신저 앱의 인트로 화면
- 화면의 방향을 세로와 가로로 바꿀 때마다 다르게 출력
- 기기를 사용하는 지역에 따라 메시지를 알맞는 언어로 출력
- 문자열, 이미지 등 리소스 파일 사용하기

### Study
* 앱 리소스, 플랫폼 리소스 활용
* 리소스 조건 설정 (Orientation, Locale)
* 리소스 디렉터리 (drawable, color, layout, values)
* string.xml 문자열 리소스 
* XML로 작성한 이미지 리소스
